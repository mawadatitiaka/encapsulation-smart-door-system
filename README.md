# encapsulation-smart-door-system
This repository contains codes and explanation to how encapsulation was used to create a smart door system for UMaT lecturers.
Encapsulation is defined as the bundling of data(attributes) and methods which operate on the data into a sinle unit(class) while restricting access to internal details.
What this means with respect to the smart door is that, every lecturer would have their secret code and no one outside the lecturers would be able to access the door except he/she gets one of the codes.
